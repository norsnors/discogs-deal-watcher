Where should the dashboard read deals from? Local scan works with just your Discogs token (no cloud). The other modes are only for when you run your own cloud watcher.

☁ 24/7 email alerts

Get deal emails around the clock — even when this app is closed. The app creates your own free copy of the watcher on GitHub and switches it on for you. You need two free accounts: GitHub (runs the scans) and Resend (sends the emails). One-time, ~5 minutes.

Create the token (pre-filled, just press Generate) →
Sign in to GitHub first; the link opens a new-token page with the right access ("repo" + "workflow") already ticked.

Get a free Resend API key →
Sign up at resend.com (Google login works), then API Keys → Create API key. Use the same email address above as you signed up with — Resend's free sender only delivers to your own address.

  1. Check GitHub + Resend access
  2. Create your own copy of the watcher
  3. Store your settings (encrypted)
  4. Switch it on + start the first scan

✈ Telegram alerts

Get every deal and 💎 rare-gem pushed straight to Telegram — instant on your phone, and it can't land in a spam folder. It runs alongside your email alerts as a backup (and fires even if an email is delayed). Free, ~2 minutes.

  1. In Telegram, open @BotFather → send /newbot → pick a name. It replies with a bot token.
  2. Open your new bot and tap Start (a bot can't message you until you do).
  3. Paste the token below and press Test — we find your chat and send a test message.

Open @BotFather in Telegram →

Saved encrypted to your own cloud watcher so alerts keep coming when this app is closed.

  1. Check GitHub access
  2. Save your Telegram settings (encrypted)
  3. Apply it + start a run
